2.0 : Tutorials

2.1 : Getting Started

When working with Matrixonator, you should see the following window:



    What do those numbers mean?

  • (1) Top Menu
    • File -> Close Option to Close Matrixonator
  • (2) List of matrices Matrixonator has.
  • (3) List of properties and options for currently selected Matrix in (2)
    • (4) Matrix details, Name, Row and Column count and Date created
    • (5) Show data button; when clicked will display the data stored in the currently selected matrix
    • (6) Matrix properties which Matrixonator can calculate
    • (7) Matrix option buttons
      • New -> Create New Matrix and add to list
      • Edit -> Modify matrix name
      • Delete -> Removes the Matrix from list and gives option to remove some computer if has been saved
      • Save -> Save matrix to computer for use everytime Matrixonator is run

    2.2 : Creating a New Matrix

    To create a new Matirx, click on the New button located at number (7) on the above image.



    Once clicked you should be presented with a window like below. Enter the name and number of rows and columns of the matrix. When have entered those details click Next to move onto next step. Click Cancel at any time to return and no new matrix will be added.



    After clicking next, you'll be asked to enter data into the rows and columns of the Matrix. Say you wished to have a matrix with 4 rows and 2 columns, you'll see something looking like this:



    Fill out each of the boxes with a value. You must enter a value into each box to continue!. If you wish to add or remove rows or columns simply press Previous and modify the values in the window shown above. Once you've entered all the data, click Next

    Your window will look something like this before clicking Next



    Once you've pressed next you should see a nice information message saying your matrix was created. Congratulations, you matrix is now appearing on the list and we can perform some calculations!
