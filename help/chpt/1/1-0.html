1.0 : Overview

1.1 : What is Matrixonator?

The Matrixonator is a mathematical Matrix calculator. It supports many different functions on matrices, listed below:

  • Reduced Row Echelon Form
  • Derminant
  • Trace
  • Inverse
  • Cofactor
Matrices can be stored to your computer. They can be added, editted and deleted at will.

1.1 : What is Matrix?

In mathematics, a matrix is a rectangular array of numbers, symbols, or expressions, arranged in rows and columns. The individual items in a matrix are called its elements or entries. Each element of a matrix is often denoted by a variable with two subscripts. For instance; a2,1 represents the element at the second row and first column of a matrix A.
